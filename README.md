# Sample SMB Solutions

A repository for the US SMB Solutions Architect Team to share architecture samples and solutions.

## Projects

- **aws-dms-mcp-server**: MCP server for AWS Database Migration Service operations
- **aws-dms-troubleshoot-mcp-server**: MCP server for AWS DMS troubleshooting and root cause analysis
- **serverless-v2-predictive-scaling**: Predictive scaling solution for AWS Serverless v2
